# 使い方
- 同じディレクトリに.envファイルを作成
- .example.envを.envに貼り付ける
- .envの中に各設定を記述する
- npm install を実行
- npm start を実行
# Config
- admin: 管理者のユーザーID
- prefix: コマンドの先頭になる文字(一部のみに有効)
- log: ログを送るチャンネルID
- error: エラーメッセージを送るチャンネルID
